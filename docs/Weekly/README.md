# Weekly Submissions and Leader Consolidation

This folder stores the weekly progress submissions of Group 54 interns and the consolidated reports compiled by the Group Leader for the Team Lead.

## Workflow

### 1. Intern Submissions (By Thursday EOD)
Each of the 7 interns should:
- Copy the [weekly_summary_template.md](file:///c:/Users/arsal/Desktop/safex/docs/Weekly/weekly_summary_template.md) template.
- Save it as `Week_[X]_[InternName].md` in this directory.
- Fill out the details accurately.

### 2. Group Leader Consolidation (By Friday 12:00 PM)
As Group Leader (Arsalan Qasim):
- Read the **Section 5 (Summary for Group Leader)** of each intern's report.
- Compile a consolidated weekly summary listing:
  - High-level achievements.
  - Development tracks progress (KB Loading, Web-UI, Similarity engine, Testing).
  - Consolidated blockers (if any).
- Save the consolidated report as `Week_[X]_Consolidated_Summary.md` in this folder.
- Send the summary to your Team Lead by Friday afternoon.
